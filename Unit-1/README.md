# Unit One
## Lesson 1 [What is a simplest C program?](Unit-1/Lesson-1.md) 最简单的C程序
### 基本概念
* C 程序 Language
	- 库函数 Library
* 编译器 Compiler
	- 链接器 Linker
* 操作系统 OS
	- 加载器 Loader

### 知识点
* 数据类型 int 
* 函数 main 
* 函数参数 void 
* 函数返回值 return

### 扩展
* 注释 comment /* */, //
* 更多数据类型 char, float 
* 变量 variable
* 局部变量 local
* 全局变量 global
