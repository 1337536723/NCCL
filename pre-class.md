# 课前准备

* 以下活动均采用 YY 语音交流

## 自我介绍 (30 mins)

* 自我介绍要点 (1 min/person)
	- 姓名，年龄，所在城市，学习/工作背景
	- 目前C语言掌握情况
	- 未来学习目标
	

## 学员分组 (10 mins)
* 2人一组，组长负责对外代表小组在班里发言，组员可以在组内发言提交问题给组长
* 组长和组员的角色，可以学习2周后互换一次
